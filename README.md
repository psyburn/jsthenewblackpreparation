Tools
=========
Node.js has to be installed on the system (with npm)

[Grunt](http://gruntjs.com/getting-started)
```sh
npm install -g grunt-cli
```

Some of the commands you'll need:

`grunt`
`grunt serve`
`grunt build`
`grunt bower:install`

[Yeoman](http://yeoman.io/)
```sh
npm install -g yo
```

Some of the commands you'll need:

`yo ...`


[Bower](http://bower.io/)
```sh
npm install -g bower
```

Some of the commands you'll need:

`bower search ...`
`bower install ...`
`bower list ...`
